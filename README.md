 # <center> List </center>

 ### Installation

 Npm: 
 ```shell
 npm install @anishshobith/list
 ```

 Yarn:
 ```shell
 yarn add @anishshobith/list
 ```

### Usage

JavaScript:
```js
const { List } = require('@anishshobith/list');

const list = new List(); // List<unknown>
```

TypeScript:
```ts
import { List } from '@anishshobith/list');

// or

import List from '@anishshobith/list';

const list = new List<V>(); // List<V>;
```
